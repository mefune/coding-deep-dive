# Concat:

-

## Example:
### *Code:*

```SQL

select   concat(column_1, ' - ', column_2) merged_columns_1_and_2
from     `xyz`
```

### *Output:*

|column_1|column_2|merged_columns_1_and_2
|---|---|---|
| 1 | 2 | 1 - 2 |
| 1 | 2 | 1 - 2 |
| 1 | 2 | 1 - 2 |
| 1 | 2 | 1 - 2 |
| 1 | 2 | 1 - 2 |
